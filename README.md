HR Attrition Analysis Using Excel & Power BI
📌 Project Overview
This project analyzes employee attrition to help organizations understand why employees leave and how to improve retention. Using Excel for data cleaning and Power BI for visualization, the project uncovers key drivers of churn and provides actionable HR recommendations.

🎯 Problem Statement
Employee attrition impacts productivity, hiring costs, and organizational stability.
The goal of this project was to:

Identify factors contributing to employee churn

Highlight high‑risk employee groups

Provide insights to support HR decision‑making

🛠️ Approach & Tools
Excel (Data Cleaning & Preparation)
Cleaned raw HR data

Handled missing values and inconsistencies

Created Salary Bands and Tenure Groups for deeper segmentation

Power BI (Dashboard & KPIs)
Built interactive dashboards with filters

Designed KPIs: Attrition Rate, Overtime %, Salary Band Churn, Department‑wise Attrition

Used DAX for custom measures and calculations

🔍 Key Insights
Sales department had the highest attrition (~20%)

Low salary band employees churned the most (~28%)

Early tenure (0–2 years) employees were more likely to leave

Overtime was a strong driver of attrition

💡 Recommendations
Improve compensation for low salary groups

Reduce overtime workload

Strengthen early‑stage engagement and onboarding

Promote work‑life balance initiatives

📈 Outcome
This project strengthened skills in data cleaning, feature engineering, KPI modeling, and dashboard storytelling. It demonstrates how HR data can be transformed into insights that support strategic retention planning.
